<h2>Переменные и константы</h2>

Для хранения данных в програме используют переменные.
Переменная это именнованная область памяти, в которой хранится значение опредленного типа. Тип определяет, какого рода информацию может хранить переменная.

` тип имя_переменной = значение;` - инициализация переменной <br>
` тип имя_переменной;` - определение переменной

Таким образом, объявление переменной - это указание типа и имени переменной. Инициализация при объявлении - указание начального значения переменной.

**Константы** 

Контанта обязательно инициализируется при объявлении, значение в дальнейшем не может быть переопределено.

` const тип ИМЯ_КОНСТАНТЫ = значение;`

1. Что выведет данный код:
``` csharp
string name = "Tom";
Console.WriteLine(Name);
```
<details>
  <summary>Ответ</summary>
  Будет ошибка. C# регистрозависим. name != Name.
</details>

***

2. Что выведет на консоль следующий код:
``` csharp
string person = "Grisha";
person = "Ann";
Console.WriteLine(person);
```
<details>
  <summary>Ответ</summary>
  Значение переменной будет перезаписано. Вывод >> Ann
</details>

***

3.Какие из следующих вариантов представляют корректное определение переменных:
```csharp

    2) person = "Tom";
    3) string person;
    4) string "Tom";
```
<details>
  <summary>Ответ</summary>
  Варианты 1) и 3)
</details>

5. Какое главное отличие инициализации от объявления переменных?
<details>
  <summary>Ответ</summary>
  При инициализации указывается значение переменной, а при объявлении только тип и имя переменной.
</details>
