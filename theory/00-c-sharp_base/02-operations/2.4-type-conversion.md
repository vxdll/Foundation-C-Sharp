<h1> Преобразования базовых типов данных </h1>

Операция преобразования типов предполагает указание в скобках того типа, к которому надо преобразовать значение:
`(тип_данных_в_который_надо_преобразовать)значение_для_преобразования;`


Ситуация:
``` csharp
byte a = 5;
int b = a + 65; // Рез-тат 70

// Но, если вместо int указать byte

byte a = 5;
byte b = a + 65; // Ошибка
```

Дело в том, что операция сложения (вычитания) возвращает значение типа int, если в операции участвуют целочисленные типы данных с разрядностью меньше или равно int (то есть типы byte, short, int).
Поэтому результатом операции a + 65 будет объект, который имеет длину в памяти 4 байта. Затем этот объект мы пытаемся присвоить переменной b, которая имеет тип byte и в памяти занимает 1 байт.

Именно в данной ситуации и необходимо преобразование типов.

``` csharp
byte a = 5;
byte b = (byte)(a+65);
```

<h2>Сужающие (narrowing) и расширяющие (widening) преобразования</h2>

Расширяющие преобразования расширяют размер объекта в памяти. Например:
``` csharp
byte a = 4;             // 0000100
ushort b = a;   // 000000000000100
```
Сужающие преобразования сужают значение до типа меньшей разрядности. Например:
``` csharp
ushort a = 4; // 2 байта
byte b = (byte) a; // сжали до 1 байта
```

<h2>Явные и неявные преобразования</h2>

1. Неявные преобразования (implicit conversion)
Компилятор сам выполняет преобразование данных вместо нас.
Механика проста. Последний бит является знаковым (0 для положительных и 1 для отрицательных чисел). При расширении в добавленные разряды копируеются знаковый бит.

``` csharp
// Для +
sbyte x1 = 4; // 0000_0100
short x2 = x1; // 0000_0000 0000_0100

// Для -
sbyte x1 = -4; // 1111_1100
short x2 = x1; // 1111_1111 1111_1100
```

2. Явные преобразования (explicit conversion)
Мы сами прописываем преобразование через конструкцию:
`(тип_данных_в_который_надо_преобразовать)значение_для_преобразования;`


| Тип | Безопасно преобразуется 
| :--- | :--- | 
| byte | short, ushort, int, uint, long, ulong, float, double, decima
| sbyte | short, int, long, float, double, decimal 
| short| int, long, float, double, decimal
| ushort| int, uint, long, ulong, float, double, decimal
| int | long, float, double, decimal
| uint| long, ulong, float, double, decimal
| long | float, double, decimal
| ulong | float, double, decimal
| float| double
| char | ushort, int, uint, long, ulong, float, double, decimal


**В остальных случаях следует использовать явные преобразования типов!
Также значения double нужно приводить к типу decimal**

``` csharp
double a = 10.8;
decimal b = (decimal)a;
```

**Не стоит забывать про потерю точности данных например из int в byte**
