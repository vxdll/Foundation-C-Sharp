<h1>Подразрядные операции</h1>

`Подразрядные операции (битовые) - действия, выполняемые над отдельными битами(0 или 1) целых чисел в их двоичном представлении.`

Числа рассматриваются в двоичном представлении:
2 -> 10 (2 разряда)
7 -> 111 (3 разряда)

<h2>Логические операции</h2>

1. & - логическое умножение

Поразрядное умноженение, если у обоих операндов значения разрядов равно 1, то операция возвращает 1, иначе 0.
``` csharp
int x = 2; // 010
int y = 5; // 101

Console.WriteLine(x&y); // 0

int a = 4; // 100
int b = 5; // 101

Console.WriteLine(a&b); // 100 или 4 в десятичном формате
// (1*1) (0*0) (0*1) -> 100
```

***

2. | - логическое сложение

Возвращается единица, если хотя бы у одного числа в данном разряде имеется единица.

``` csharp
int a = 2; // 010
int b = 5; //101

Console.WriteLine(a|b); // 111 или 7 (десятичный формат)
// (1|1) (1|0) (0|1) -> 111
```

***

3. ^ - исключающее ИЛИ

Операция XOR, нередко ее применяют для простого шифрования:

``` csharp
int a = 90; // Шифруемое значение. 01011010
int b = 152; // Ключ. 10011000

int encrypt = a ^ b; // 11000010 или 194 (десятичный формат)

/* Если биты различны, то рез-тат = 1, иначе 0 */ 

```
4. ~ - логическое отрицание или инверсия

Инвертирует все разряды: если значение разряда равно 1, то оно становится равным нулю, и наоборот.

``` csharp
int x = 12; // 00001100
Console.Write(~x); // 11110011
```

<h2>Представление отрицательных чисел</h2>

Two’s complement - дополнительный код для записи чисел со знаком С#. Старший разряд является знаковым.

Алгоритм перевода:
``` csharp
int x = 12; // Исходное число

//1 Инвертировать число
int y = ~x;

//2 Прибавить 1
y += 1;

Console.WriteLine(y); // -12
```

<h2>Операции сдвига</h2>

``` csharp
/*
Два варианта:
1. x<<y - сдвигает число x влево на y разрядов. Например, 4<<1 сдвигает число 4 (которое в двоичном представлении 100) на один разряд влево, то есть в итоге получается 1000 или число 8 в десятичном представлении.
2. x>>y - сдвигает число x вправо на y разрядов. Например, 16>>1 сдвигает число 16 (которое в двоичном представлении 10000) на один разряд вправо, то есть в итоге получается 1000 или число 8 в десятичном представлении.
Таким образом, если исходное число, которое надо сдвинуть в ту или другую строну, делится на два, то фактически получается умножение или деление на два. Поэтому подобную операцию можно использовать вместо непосредственного умножения или деления на два. Например:
*/

int a = 16; // в двоичной форме 10000
int b = 2; // в двоичной форме
int c = a << b; // Сдвиг числа 10000 влево на 2 разряда, равно 1000000 или 64 в десятичной системе
 
Console.WriteLine($"Зашифрованное число: {c}") ;    // 64
 
int d = a >> b; // Сдвиг числа 10000 вправо на 2 разряда, равно 100 или 4 в десятичной системе
Console.WriteLine($"Зашифрованное число: {d}");     // 4
```


<h3> Практический пример сдвига операций </h3>


``` csharp

/* Условие задачи
Есть 4 маленьких числа (от 0 до 15), и ты хочешь сохранить их в одном числе, чтобы сэкономить память или передать по сети.
*/

// Исходные данные (каждое от 0 до 15)
int a = 3;   // первое число
int b = 7;   // второе число
int c = 12;  // третье число
int d = 1;   // четвёртое число
        
Console.WriteLine("=== Исходные числа ===");
Console.WriteLine($"a = {a}, b = {b}, c = {c}, d = {d}");
Console.WriteLine();
        
/* ========== УПАКОВКА ==========
Каждое число занимает 4 бита (от 0 до 15)
Сдвигаем каждое число на своё место и складываем
*/

int packed = 0;
        
packed = packed | (a << 0);   // a на позицию 0 (биты 0-3)
packed = packed | (b << 4);   // b на позицию 4 (биты 4-7)
packed = packed | (c << 8);   // c на позицию 8 (биты 8-11)
packed = packed | (d << 12);  // d на позицию 12 (биты 12-15)

Console.WriteLine("=== Упакованное число ===");
Console.WriteLine($"packed = {packed}");
Console.WriteLine($"в двоичном виде: {Convert.ToString(packed, 2).PadLeft(16, '0')}");
Console.WriteLine();
        
/* ========== РАСПАКОВКА ==========
Достаём каждое число обратно */
        
int a2 = (packed >> 0) & 0b1111;   // сдвиг на 0, маска 4 бита
int b2 = (packed >> 4) & 0b1111;   // сдвиг на 4, маска 4 бита
int c2 = (packed >> 8) & 0b1111;   // сдвиг на 8, маска 4 бита
int d2 = (packed >> 12) & 0b1111;  // сдвиг на 12, маска 4 бита
        
Console.WriteLine("=== Распакованные числа ===");
Console.WriteLine($"a = {a2}, b = {b2}, c = {c2}, d = {d2}");
Console.WriteLine();
        
// Проверка
Console.WriteLine($"=== Проверка ===");
Console.WriteLine($"Упаковка/распаковка прошла успешно: {(a == a2 && b == b2 && c == c2 && d == d2)}");
```
