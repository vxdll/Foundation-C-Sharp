<h2>Базовые арифметические операции</h3>

Операции - это действия над операндами.
Операнд - переменная или како-либо значение (например число).

Операции:
1. Унарные - 1 операнд.
2. Бинарные - 2 операнда.
3. Тернарные - 3 операнда.

<h3>Бинарные арифметические операции</h3>

1. Сложение
``` csharp
int x = 10;
int y = 5;
int result = x + y; // 15
```

***

2. Вычитание
``` csharp
int x = 10;
int y = 5;
int result = x - y; // 5
```
***

3. Умножение
``` csharp
int x = 10;
int y = 5;
int result = x * y; // 50
```
***

4. Деление
``` csharp
int x = 10;
int y = 5;
int result = x / y; // 2
```
***

**Важно!**
При делении если оба операнда целые числа, то рез-тат будет округляться до целого числа.

``` csharp
int x = 10;
int y = 5;
double result = x / y; // 5
```

 ***

5. Остаток от деления %
``` csharp
int x = 10;
int y = 4;
int result = x % y; // 2
```
***

<h3>Унарные операции</h3>

1. Инкремент ++
``` csharp
/* Префиксный ++x - значение увеличивается на 1, а потом значнеие возвращается в кач-ве рез-та.
Постфиксный x++ - значение переменной возвращается в качетве рез-та операции, а затем к нему прибавляется 1
*/

int x1 = 5;
int z1 = ++x1; // z1=6; x1=6
Console.WriteLine($"{x1} - {z1}");
 
int x2 = 5;
int z2 = x2++; // z2=5; x2=6
Console.WriteLine($"{x2} - {z2}");

```
1. Декремент --
``` csharp
/* Префиксный --x - значение уменьшается на 1, а потом значнеие возвращается в кач-ве рез-та.
Постфиксный x-- - значение переменной возвращается в качетве рез-та операции, а затем уменьшается на 1
*/

int x1 = 5;
int z1 = --x1; // z1=4; x1=4
Console.WriteLine($"{x1} - {z1}");
 
int x2 = 5;
int z2 = x2--; // z2=5; x2=4
Console.WriteLine($"{x2} - {z2}");
```

<h3>Порядок выполнения арифметических операций</h3>

